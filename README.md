This is my Everybody Loves Trains Project

To run the code, I've been copying and pasting the code into TypeScript Playground.
All 7 problems are solved in the file titles trains.ts, to run it, simply copy and paste the entire file into TypeScript Playground. 

Please note, each of the main functions have a console log which logs what the function returns. However, these console logs are commented 
out for problems 1-3. The reason is because the console logs slowed down the run time too significantly when testing problems 4-7. 
So, when testing what the functions for problems 1-3 return, uncomment the log statements that are on lines 278, 339, and 374. But
make sure to comment these statements out again before running the full file or the code for problems 4-7.
